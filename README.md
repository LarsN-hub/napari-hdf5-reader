## Installation

You can install `napari-hdf5-reader` via [pip]:

```bash
pip install napari-hdf5-reader
```

If napari is not already installed, you can install `napari-hdf5-reader` with napari and Qt via:

```bash
pip install "napari-hdf5-reader[all]"
```
